Sumatra PDF is a PDF, ePub, MOBI, CHM, XPS, DjVu, CBZ, CBR reader for Windows


File menu

  • Ctrl+o - Open
  • Ctrl+w - Close
  • Ctrl+s - Save As
  • F2 - Rename
  • Ctrl+p - Print
  • Ctrl+d - Properties
  • Ctrl+q - Quit

Go To menu

  • Right Arrow - Next Page
  • Left Arrow - Previous Page
  • Home - First Page
  • End - Last Page
  • Ctrl+G - Page...
  • Alt+Left Arrow - Back
  • Alt+Right Arrow - Forward
  • Ctrl+F - Find...

View menu

  • Ctrl+6 - Single Page
  • Ctrl+7 - Facing
  • Ctrl+8 - Book View
  • Ctrl+Shift+- - Rotate Left
  • Ctrl+Shirt++ - Rotate Right
  • F5 - Presentation
  • F11 - Fullscreen
  • F12 - Bookmarks
  • Ctrl+A - Select All
  • Ctrl+C - Copy Selection

Zoom menu

  • Ctrl+0 - Fit Page
  • Ctrl+1 - Actual Size
  • Ctrl+2 - Fit Width
  • Ctrl+3 - Fit Content
  • Ctrl+Y + Custom Zoom...
There is a table of these shortcuts posted at https://www.sumatrapdfreader.org/manual.html

Navigation

  • j/k - scroll up/down by line
  • Up/Down - scroll up/down by line
  • space - scroll by screen
  • <Shift> + space - scroll back by screen
  • n/p - next/previous page
  • Page Down/Page Up - next/previous page
  • <Ctrl> + Down/<Ctrl> + Up - next/previous page
  • <Alt> + Left - go back
  • <Alt> + Right - go forward
  • <Ctrl> + G - go to page
  • g - go to page
  • Home - go to first page
  • End - go to last page
  • b - flip a page in book mode
  • <Ctrl> + <Shift> + Right - open next document in the directory
  • <Ctrl> + <Shift> + Left - open previous document in the directory

Viewing state

  • +, - - zoom in/out
  • <Ctrl> + scroll wheel - zoom in/out
  • z - toggle zoom between Fit Page, Fit Width, Fit Content
  • c - toggle between continous/non-continous mode
  • <Ctrl> <Shift> - - rotate left
  • / on numeric keypad - rotate left
  • <Ctrl> <Shift> + - rotate right
  • * on numeric keypad - rotate left
  • F12 - show/hide bookmarks (table of contents)
  • F6 - switch focus between bookmarks window and main window
  • <Ctrl> + L - enter presentation mode (minimal full screen mode)
  • F11 - toggle presentation mode (minimal full screen mode)
  • <Shift> + F11 enter full screen mode (black border)
  • <Ctrl> <Shift> + L - toggle full screen mode (grey border)
  • ESC - exit full screen or presentation mode
  • . - in presentation mode, change background to black
  • w - in presentation mode, change background to white
  • i - show page information in full screen/presentation mode
  • m - show cursor position in document coordinates
  • F8 - show/hide toolbar
  • F9 - show/hide menu

Actions

  • <Ctrl> + O - open a new document
  • <Ctrl> + W - close current document
  • <Ctrl> + F4 - also close current document
  • <Ctrl> + S - save as
  • <Ctrl> + P - print
  • r - reload
  • <Ctrl> + F - find text
  • / - find text
  • F3 - find next
  • <Shift> + F3 - find previous
  • <Ctrl> + Q - quit program
  • q - quit program
  • <Ctrl> + Left Mouse - select text or image

Tabs

  • <Ctrl> + Tab - go to next tab
  • <Ctrl> <Shift> + Tab - go to previous tab
  • <Alt> + 1 - go to tab 1. Use 2, 3 etc. up to 8 to go to n-th tab
  • <Alt> + 9 - go to last tab
