SumatraPDF is a multi-format (PDF, EPUB, MOBI, FB2, CHM, XPS, DjVu) reader
for Windows under (A)GPLv3 license, with some code under BSD license (see
AUTHORS).

Downloads and documentation: http://www.sumatrapdfreader.org

More info about the project: https://github.com/sumatrapdfreader/sumatrapdf/wiki

To compile you need Visual Studio 2015 (free Community edition from
https://www.visualstudio.com/products/visual-studio-community-vs will work).

Open vs2015/SumatraPDF.sln.
